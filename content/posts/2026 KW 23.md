- Wirklich viel das neue *Boards of Canada* Album gehört - ich kann es wärmstens empfehlen.

- [Language Transfer](https://www.languagetransfer.org/complete-spanish) für Spanisch angefangen. Das Konzept gefällt mir, mal sehen, was so hängen bleibt.  
- Weiter an Knausgaard rumgelesen. Das wird gerade wirklich zäh. Eigentlich wurde der sechste Band der Reihe als der Beste genannt, aber das kann ich aktuell so nicht bestätigen.  
- Eine [Mealie-Instanz](https://mealie.io/) auf dem Homeserver aufgesetzt und alle meine Rezepte aus Obsidian via Claude importiert. Das ging echt sehr easy vonstatten. Mealie macht die Essensplanung für mich einfacher, da ich via REST API irgendwelche Rezepte reinpumpen kann, egal aus welcher Quelle, ob das jetzt seriouseats oder irgendein anderer Essensblog ist. Claude macht das natürlich noch einfacher, in dem es sogar Rezepte aus Fotos aus Kochbüchern lesen kann und die Zutaten dann richtig zuordnet. So wird aus einem Wochenplan eine klare Einkaufsliste, die sogar nach den Abteilungen im Supermarkt sortiert ist. Klare Empfehlung für alle Selfhoster da draußen!
